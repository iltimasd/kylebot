Introducing Kylebot: the advanced rainbow text

generator~Kylebot is a GUI application that allows

users to create a color gradient through text.

The app is avaible for OSX/macOS, Windows 32-bit, Windows 64-bit, Linux, and Linux(ARM) --> https://goo.gl/Mj7t7W

You can check out the repo for the app --> https://github.com/iltimasd/kylebot

~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

Here's how it works!

Step 1

Users use interface to customize their gradient

through manipulating hue, saturation, and brightness.

Step 2

Users then drag and drop text, either static text or

typed text, into the application window.

Step 3

Users finally confirm their input text and color setting

by pressing any key. The color text is now on the

user’s clipboard, ready to paste in emails, text

documents, and blog posts.

*GIF compression creates color banding; note that the application displays a finer gradient.
